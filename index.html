Elementos HTML

Un párrafo es una sección de texto que se encuentra entre dos etiquetas <p>.

Un salto de línea se logra utilizando la etiqueta <br>
como se muestra aquí.

Cambio de tema en un párrafo utilizando <h2>.

Este es un nuevo tema

La etiqueta <wbr> se utiliza para indicar posibles lugares de corte en una palabra, por ejemplo:

Esta es una palabra larga que podría dividirse en dos líneas.

Para crear superíndices y subíndices, puedes utilizar las etiquetas <sup> y <sub>:

Agua es H2O.

El número π es aproximadamente 3.14159 x 107.

Para resaltar texto importante, se utiliza la etiqueta <strong>:

Este texto es importante.

El énfasis se logra con la etiqueta <em>:

Este texto tiene énfasis.

Una cita en línea se crea con la etiqueta <q>:

El autor dijo: La educación es el arma más poderosa que puedes usar para cambiar el mundo.

Una cita en bloque se crea con la etiqueta <blockquote>:

La ciencia es el proceso que nos ayuda a entender cómo funciona el mundo y a hacer preguntas cada vez más precisas sobre él.

- Carl Sagan

La etiqueta <cite> se utiliza para citar la fuente de una obra:

Este libro fue escrito por John Doe.

La etiqueta <dfn> se usa para definir términos:

HTML significa HyperText Markup Language.
